i can’t find a rhyme for orange,
so i’ll lock it in a box.

i can’t find a rhyme for silver,
so i’ll steal all but it’s socks.

i can’t find a rhyme for rhythm,
so i’ll break apart its town.

i can’t find a rhyme for gurgle,
so i’ll spin it all around.

i can’t find a rhyme for chamber,
so i’ll pigment a page.

i can’t find a rhyme for nothing,
so i’ll smash it in a rage.
